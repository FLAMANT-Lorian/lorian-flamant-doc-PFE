Améliorations JavaScript et dégradation gracieuse

Filtre et recherche en AJAX

Afin d’améliorer l’expérience utilisateur, une fonctionnalité AJAX a été mise en place pour la barre de recherche ainsi que pour le système de filtre. Cette approche permet de mettre à jour les résultats sans recharger la page, offrant une navigation plus fluide à l’utilisateur.

Une attention particulière a toutefois été portée à la dégradation gracieuse. Si JavaScript est désactivé, ces fonctionnalités restent utilisables grâce à un formulaire classique. Dans ce cas, une soumission du formulaire est nécessaire pour effectuer la recherche ou appliquer les filtres.

Lorsque JavaScript est activé, l’événement de soumission du formulaire est intercepté à l’aide de preventDefault(). Cela permet d’empêcher l’envoi classique de la requête HTTP du formulaire et d’éviter l’exécution simultanée d’une requête côté serveur et d’une requête AJAX.

Fancybox

Sur le site public, les images peuvent être affichées en grand grâce à l’utilisation de Fancybox. Pour permettre cela, chaque image est encapsulée dans une balise a dont l’attribut href pointe vers l’image originale.

Cette fonctionnalité respecte également le principe de dégradation gracieuse. Si JavaScript est désactivé, le lien continue de fonctionner de manière classique et ouvre l’image directement dans le navigateur. Lorsque JavaScript est activé, Fancybox intercepte ce lien et affiche l’image dans une modale directement dans la page. L’utilisateur peut ainsi consulter les images sans quitter la page en cours, tout en conservant un comportement fonctionnel lorsque JavaScript n’est pas disponible.

Calendrier

Sur la page de présentation de la salle, un calendrier permet d’indiquer à l’utilisateur les dates encore disponibles à la réservation. Celui-ci est généré à l’aide d’un package JavaScript et nécessite donc JavaScript pour être affiché. En l’absence de JavaScript, le calendrier n’est pas disponible. Même si JavaScript n'est pas présent, le site continue de fonctionner normalement.

Carroussel

Sur certaines pages du site public, un carrousel est utilisé pour présenter différents actualités. Son fonctionnement est amélioré à l’aide de JavaScript afin d’offrir des contrôles de navigation et une expérience plus confortable à l’utilisateur.

En l’absence de JavaScript, le carrousel reste accessible. L’utilisateur peut parcourir les différentes informations à l’aide du défilement horizontal natif du navigateur. Lorsque JavaScript est activé, les fonctionnalités avancées du carrousel sont ajoutées pour rendre l'expérience de l’utilisateur plus meilleure.

Animations

Les animations du site public ont été réalisées à l’aide de la bibliothèque GSAP. Celles-ci sont entièrement construites en JavaScript et viennent ajouter du dynamisme à l’interface.

En l’absence de JavaScript, les animations ne sont tout simplement pas jouées. Le contenu et les fonctionnalités du site restent fonctionnelles, garantissant ainsi le bon fonctionnement du site aux utilisateurs ne possédant pas JavaScript.

Finalement, une attention particulière a été portée à l’accessibilité en prenant en compte la préférence prefers-reduced-motion. Lorsque cette option est activée au niveau du système d’exploitation ou du navigateur, les animations sont désactivées afin de limiter les effets de mouvement pouvant gêner l’utilisateur. Cette approche permet de garantir une expérience optimale pour tous les utilisateurs du site.
