Méthodologie de mon analyse

L’analyse s’est basée sur les 23 catégories (14 thématiques, 3 phases et 6 mots clés) de l’audit Opquast. Parmi toutes ces règles, deux correspondantes au projet ont été sélectionnées par catégorie.

ID : 2 Catégorie : E-commerce Règles : – Statut : Non concerné Description : –
ID : 3 Catégorie : Identification et contact Règles : Règle 101 – Règle 107 Statut : Conforme Description : Identité de l’ASBL indiquée dans le footer – Différents moyens de contact : Téléphone, email

ID : 6 Catégorie : Présentation Règles : Règle 182 – Règle 194 Statut : Conforme Description : Tous les contrastes ont été vérifiés – Site consultable sur toutes les tailles d’écrans existantes
